Impact Martial Arts Belt Tracker

Dashboard

The Dashboard shows a quick summary of all students and their belts.

  • Belt Counts: See how many students hold each belt color.
  • No students yet? Add students using the Students section below.

Students

Manage all student records in this section.

  • Add Student: Click Add Student to enter a new student.
  • Required fields: Name, Birthdate, Belt.
  • Optional: Instructor, Location, Eligible for Belt Test, Active status.
  • Edit or delete students using the buttons in the Actions column.
  • Age is calculated automatically from the birthdate.

Belt Tests

Track studentsâ€™ belt tests.

  • Add Belt Test: Click Add Belt Test to record a test.
  • Select eligible and active students only.
  • Enter Belt, Date, Instructor(s), and Location.
  • After saving, the student's belt is automatically updated.
  • Edit or delete belt tests using the buttons in the Actions column.

Control Panel

  • Manage Belts: Add, edit, or delete belt names.
  • Manage Instructors: Add or remove instructors.
  • Manage Locations: Add or remove training locations.
  • Import Data: Load previously exported data from a JSON file.
  • Export Data: Download all data as a JSON file for backup or transfer.
  • Reset Data: Clear all data and reset to default belts.

Tips & Notes

  • Active students are highlighted in green in the Students table.
  • Eligibility checkbox determines if a student can be added to Belt Tests.
  • Use multiple selection in instructors when a test involves more than one instructor.
  • All changes are saved automatically in local storage.
