Impact Martial Arts Belt Tracker – Instructions

Version 2.4 – Stable

This manual provides detailed guidance on how to use the Belt Tracker system, including its features, workflow, and business rules.

1. Dashboard Overview

  • The Dashboard displays:
    • Total count of students per belt.
    • Number of Active and Inactive students.
    • Number of students Eligible for belt tests.
  • The dashboard updates automatically when students or tests are added/edited.

2. Student Management

  1. Adding a Student:
    1. Click the Add Student button.
    2. Enter the Name and Birthdate. Age is calculated automatically.
    3. Select a Belt, Instructor, and Location.
    4. Check Eligible if the student is ready for a test.
    5. Check Active if the student is currently enrolled.
    6. Click Save to add the student.
  2. Editing a Student:
    1. Click Edit next to the student in the table.
    2. Update any information, including belt, instructor, location, or active status.
    3. Save changes. The table and dashboard will update automatically.
  3. Deleting a Student:
    1. Click Delete next to the student.
    2. Confirm the deletion. This removes the student and all associated belt tests.
  4. Business Rules for Students:
    • Only Active students are eligible for belt tests.
    • After a belt test, eligibility is removed and the Next Eligible Date is automatically set 3 months from the test date for active students.
    • Once the eligibility date arrives, the student becomes Eligible again.

3. Belt Tests

  1. Adding a Belt Test:
    1. Click Add Belt Test.
    2. Select a student from the dropdown. Only students who are Active and Eligible appear.
    3. Select the Belt being tested for, the Date, Instructor(s), Location, and Result (Pass/Fail).
    4. Click Save.
  2. Editing a Belt Test:
    1. Click Edit next to the test entry.
    2. Update any details, then save.
  3. Deleting a Belt Test:
    1. Click Delete next to the test entry.
    2. Confirm deletion. The student’s eligibility status will not change automatically; manual adjustment may be needed.
  4. Business Rules for Belt Tests:
    • Passing a test updates the student’s belt automatically.
    • After a test, the student becomes Ineligible until the Next Eligible Date is reached.
    • Eligibility returns automatically only if the student is Active.

4. Control Panel

  • Manage Belts: Add, remove, or reorder belt types.
  • Manage Instructors: Add or remove instructors for assignment to students and tests.
  • Manage Locations: Add or remove locations where classes/tests occur.
  • Import / Export Data: Save your system data to a JSON file or load from one.
  • Reset Data: Clears all students, tests, instructors, and locations. Belt defaults remain.

5. Search and Filter

  • Use the search bar in the Students section to filter by name, belt, instructor, or location.
  • Filtering updates the table in real-time without affecting data.

6. General Notes

  • Always save changes in modals before closing.
  • Age is automatically calculated from birthdate.
  • Students who are inactive are ignored in eligibility and belt test selection.
  • Next Eligible Date is displayed in the Students table between Eligible and Active columns.
  • The system auto-updates dashboards and eligibility status on page load and after any save action.
