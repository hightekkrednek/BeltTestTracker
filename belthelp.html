Impact Martial Arts Belt Tracker - Instructions

Overview

The Impact Martial Arts Belt Tracker is designed to help instructors manage students, belt tests, and other key information in an organized way. Features include:

  • Adding, editing, and deleting students
  • Tracking belt tests and automatically updating student belts
  • Managing instructors, belts, and locations
  • Importing and exporting data for backup
  • Displaying student counts, active/inactive status, and eligibility for belt tests

Dashboard

The dashboard shows a summary of the number of students per belt.

  • Visit the top of the page to see counts per belt automatically updated.

Students

  1. Click Add Student to open the student modal.
  2. Fill in Name, Birthdate, Belt, Instructor, Location, and check Eligible and Active if applicable.
  3. Click Save to add or update the student.
  4. The Students Table displays all students with their:
    • ID
    • Name
    • Birthdate
    • Age (calculated automatically)
    • Belt
    • Instructor
    • Location
    • Eligible status
    • Active status
  5. Edit a student by clicking Edit or remove with Delete.
  6. Above the student table, you'll see counts of Active and Inactive students.

Belt Tests

  1. Click Add Belt Test to open the test modal.
  2. Select a student (only Active and Eligible students appear).
  3. Choose the Belt being tested for, Date, Instructor(s), and Location.
  4. Click Save to record the test. The student's belt will automatically update to the new belt.
  5. The belt test table shows all test records with options to edit or delete.
  6. Above the belt test table, you'll see a count of students eligible for belt tests.

Control Panel

Located below the tables, the Control Panel lets you manage global data:

  • Manage Belts - Add, edit, or remove belt types. You can also change the order of belts by using the up (↑) and down (↓) arrows in the Belt Management modal. This order determines how belts appear in dropdown lists and reports.
  • Manage Instructors - Add or remove instructors
  • Manage Locations - Add or remove training locations
  • Import Data - Load data from a JSON file
  • Export Data - Save current data to a JSON file
  • Reset Data - Clear all data and restore default belts

Tips & Best Practices

  • Always make sure students are marked Active and Eligible to appear in belt test selections.
  • Use the import/export functions to backup your data regularly.
  • Keep belt names consistent to ensure proper tracking and dashboard accuracy.
  • Check the Dashboard for a quick visual of belt distribution.
