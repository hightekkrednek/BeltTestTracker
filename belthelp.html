Impact Martial Arts Belt Tracker – Help / Instructions

Version 2.4 – Stable

Welcome to the latest version of the Belt Tracker. This guide explains the new features, workflow, and tips for using the system efficiently.

New Features

  • Next Eligibility Date: After a belt test, the system automatically sets a 3-month wait period before the student becomes eligible again. The date appears in the Next Eligible Date column.
  • Eligibility Automation: Active students automatically regain eligibility when their scheduled date arrives.
  • Student Management: Easily add/edit students, set their active status, and track eligibility. Age is calculated from birthdate automatically.
  • Belt Test Handling: Selecting a student for a test is limited to those who are active and eligible. Passing a test updates their belt and schedules next eligibility.
  • Dashboard Updates: View counts of active, inactive, and eligible students, plus belt distribution statistics.
  • Data Management: Import/export all system data or reset to default belts and start fresh.

Quick Tips

  • Use the search bar to filter students by name, belt, instructor, or location.
  • Click Edit to modify student or belt test information.
  • Ensure a student is marked as Active to be considered for belt tests and automated eligibility scheduling.
  • Always save changes in modals before closing.
